# homebrew-tap

Personal [Homebrew](https://brew.sh) tap.

## Install

```sh
brew tap araidz/tap
brew trust araidz/tap   # third-party taps require an explicit trust
brew install --cask lidless
```

## Casks

- **lidless** — menu-bar app that keeps a Mac awake with the lid closed.
  Upstream: [nghialuong/Lidless](https://github.com/nghialuong/Lidless).
